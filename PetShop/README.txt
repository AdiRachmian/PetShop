# PetShop Project

Welcome to PetShop, a web application designed to manage and showcase animals available for adoption
in a pet store setting. This README provides an overview of the project, its features, installation steps,
usage instructions, and more.

## Features

- Browse animals by categories, names, and other filters.
- Add, edit, and delete animal listings with images.
- Leave comments on animal listings for user interaction.
- Responsive design for seamless user experience on various devices.

## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- [.NET Core SDK](https://dotnet.microsoft.com/download)
- [Entity Framework Core CLI](https://docs.microsoft.com/en-us/ef/core/cli/dotnet)

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/YourGitHubUsername/PetShop.git
    cd PetShop

2. Restore dependencies:
	dotnet restore

3. Apply database migrations:
	dotnet ef database update

4. Run the application:
	dotnet run

5. Access the app in your browser at http://localhost:5000.

## Usage

1. Browse animals on the Catalog page.
2. Add new animals with images using the "Add Animal" page.
3. Edit or delete animals from their respective detail pages.
4. Engage with the community by leaving comments on animal listings.

## Technologies Used
